Detailed Description

This is a library for the AT42QT1070 I2C 7 Key Capacitive Touch Sensor

Designed specifically to work with the Oak Development Technologies AT42QT1070 Acorn -—> urlhere

The sensor uses I2C to communicate with additional pins that can be connected to for other operational modes.
